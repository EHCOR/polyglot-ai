# Overview
Simple translation demo app using the OpenAI Responses API.
Basic Vite, Express, JS, HTMl, CSS project

# Getting Started
Create your .env
```
AI_KEY=Your API Key
AI_URL=URL to your provider (must support responses API)
AI_MODEL=your_model
PORT=backend port
NODE_ENV=production or development
```

Install the dependencies and run the project
```
npm install
npm start & npm run server (need both up)
```
